# `removeRecordingEventListener`

:::danger Removed in v2.0.0
This API has been **removed** in v2.0.0. The [`useSGScreenRecord`](./use-sg-screen-record.md) hook automatically handles cleanup when the component unmounts.
:::

---

**v1.x Documentation (for reference only)**

**(iOS only)** Deactivate all callback screen recording handlers.

## Example code (v1.x)

```js
ScreenGuardModule.removeRecordingEventListener();
```
