# `removeScreenshotEventListener`

:::danger Removed in v2.0.0
This API has been **removed** in v2.0.0. The [`useSGScreenShot`](./use-sg-screenshot.md) hook automatically handles cleanup when the component unmounts.
:::

---

**v1.x Documentation (for reference only)**

Deactivate all callback screenshot handlers.

## Example code (v1.x)

```js
ScreenGuardModule.removeScreenshotEventListener();
```
