# `removeScreenshotEventListener`

:::danger v2.0.0에서 제거됨
이 API는 v2.0.0에서 **제거**되었습니다. [`useSGScreenShot`](./use-sg-screenshot.md) 훅이 컴포넌트 언마운트 시 자동으로 정리를 처리합니다.
:::

---

**v1.x 문서 (참조용)**

모든 스크린샷 콜백 핸들러를 비활성화합니다.

## 예제 코드 (v1.x)

```js
ScreenGuardModule.removeScreenshotEventListener();
```
