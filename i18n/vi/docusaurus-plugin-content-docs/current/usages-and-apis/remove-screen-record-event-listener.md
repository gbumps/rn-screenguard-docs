# `removeRecordingEventListener`

:::danger Đã xóa trong v2.0.0
API này đã bị **xóa** trong v2.0.0. Hook [`useSGScreenRecord`](./use-sg-screen-record.md) tự động xử lý việc dọn dẹp khi component unmount.
:::

---

**Tài liệu v1.x (chỉ dùng để tham khảo)**

**(Chỉ dành cho iOS)** Hủy kích hoạt tất cả các trình xử lý callback quay màn hình.

## Code mẫu (v1.x)

```js
ScreenGuardModule.removeRecordingEventListener();
```
