# `removeScreenshotEventListener`

:::danger Đã xóa trong v2.0.0
API này đã bị **xóa** trong v2.0.0. Hook [`useSGScreenShot`](./use-sg-screenshot.md) tự động xử lý việc dọn dẹp khi component unmount.
:::

---

**Tài liệu v1.x (chỉ dùng để tham khảo)**

Hủy kích hoạt tất cả các trình xử lý callback chụp màn hình.

## Code mẫu (v1.x)

```js
ScreenGuardModule.removeScreenshotEventListener();
```
