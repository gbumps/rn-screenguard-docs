# `removeRecordingEventListener`

:::danger 在 v2.0.0 中已移除
此 API 已在 v2.0.0 中**移除**。[`useSGScreenRecord`](./use-sg-screen-record.md) Hook 会在组件卸载时自动处理清理工作。
:::

---

**v1.x 文档 (仅供参考)**

**(仅限 iOS)** 停用所有录屏回调处理器。

## 示例代码 (v1.x)

```js
ScreenGuardModule.removeRecordingEventListener();
```
